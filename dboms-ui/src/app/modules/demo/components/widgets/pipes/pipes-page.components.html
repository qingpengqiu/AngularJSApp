1、iq_array_refactor示例


XXX为数组[obj1,obj2] 将按照expression的模式显示数组 其中[.属性]替换数组元素的属性值 id name为可选，也可替换 如 任意1[.id]任意2[.name]任意3 显示 : 任意1+"obj1.id"+任意2+"obj1.name"+任意3,任意1+"obj2.id"+任意2+"obj2.name"+任意3 XXX 为
[ {{ '{' }}id:"id1",username:"xuchao1"{{ '}' }}, {{ '{' }}id:"id2",username:"xuchao2"{{ '}' }}, {{ '{' }}id:"id2",username:"xuchao2"{{ '}' }}
若expression为[.username]([.id])，显示结果为：{{ users | iq_array_refactor:"[.id]([.username])"}}

{{ '{' }}{{ '{' }}XXX | iq_array_refactor:"[.id][.name]" {{ '}' }}{{ '}' }}

若表达式为： 
{{ '{' }}{{ '{' }}XXX | iq_array_refactor:"[.id]([.name])" {{ '}' }}{{ '}' }}

显示结果为： {{ users | iq_array_refactor:"[.id]([.username])"}}
若表达式为： 
{{ '{' }}{{ '{' }}XXX | iq_array_refactor:"[.id]---[.name]" {{ '}' }}{{ '}' }}

显示结果为： {{ users | iq_array_refactor:"[.id]---[.username]"}}
若表达式为： 
{{ '{' }}{{ '{' }}XXX | iq_array_refactor:"[.id]" {{ '}' }}{{ '}' }}

显示结果为： {{ users | iq_array_refactor:"[.id]"}}
若表达式为： 
{{ '{' }}{{ '{' }}XXX | iq_array_refactor:"[.name]" {{ '}' }}{{ '}' }}

显示结果为： {{ users | iq_array_refactor:"[.username]"}}
