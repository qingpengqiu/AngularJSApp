tab-switch示例

tab切换组件

tab列表由组件内部操作，内容列表由父组件同步操作
tabList,传进tab的内容格式为[ {{'{'}} text:'',invalid:Boolean {{'}'}},... ]
active,传一开始呈现tab的index
confirmText,传点击tab删除时提示的文字
canAdd,布尔值，表示该tab组件是否支持增加tab,默认为false
canDelete，布尔值，表示该tab组件是否支持删除tab,默认为false
onChange事件,当切换tab显示时，返回当前显示tab的index
onDelete事件，当删除一个tab时，返回删除tab的index
addOne事件，当增加一个tab时，返回增加tab的index，新增加tab的invalid默认为false

            {{'<'}}tab-switch [tabList]="xx" [active]="y" confirmText="string" canAdd="true" canDelete="true" 
                            (onChange)="onChange($event)" (onDelete)="deleteTab($event)" (addOne)="addTab($event)"{{'>'}}{{'<'}}/tab-switch{{'>'}}
